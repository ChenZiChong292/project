excel文件夹存放原始数据
fish文件夹存放预测鲢鳙鱼投放数量预测文件
train_y1.py预测鲢鱼常年平均亩放养量（千克/亩）
train_y2.py预测鳙鱼常年平均亩放养量（千克/亩）
index文件夹存放预测各项水质指标的预测文件
kind文件夹存放预测水库营养型的预测文件
weight文件夹存放各预测文件的指标权重

predict_kind.py：选取总磷、总氮、高锰酸钾和叶绿素4个指标，用水库全年网外（养鱼前）的指标值作为输入，调用方法predict_kind，可以预测出水库的营养型

predict_fish.py：选取浮游植物平均密度（104cell/L），平均植物量（mg/L），浮游动物平均密度（个/L），平均动物量（mg/L），浮游植物多样性和浮游动物多样性6个指标，用水库全年网外（养鱼前）的指标平均值作为输入，调用方法predict_fish，可以预测出水库的投鱼量

predict_index.py：选取总磷、总氮、高锰酸钾和叶绿素、浮游植物平均密度（104cell/L），平均植物量（mg/L），浮游动物平均密度（个/L），平均动物量（mg/L），浮游植物多样性和浮游动物多样性共10个指标的网外季度平均值作为输入，调用方法predict_index可以预测出养鱼后水质的变化情况
输入的季度与预测的季度相对应（输入December的数据输出也为December）

根据预测得到的投鱼量和水库营养型，可以计算出养鱼带来的间接效益和直接效益
